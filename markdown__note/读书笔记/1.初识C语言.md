# 源代码、目标代码文件、可执行文件

 |     文件     |                          解释                          |
 | :----------: | :----------------------------------------------------: |
 |  源代码文件  |                   用户编写的代码文件                   |
 | 目标代码文件 |             源代码编译成的机器语言代码文件             |
 |  可执行文件  | 目标代码+系统标准启动代码+库代码通过连接器合并成的文件 |

# 编程的七个步骤

1. 定义程序目标
2. 设计程序
3. 编写代码
4. 编译
5. 运行程序
6. 测试和调试程序
7. 维护和修改程序
